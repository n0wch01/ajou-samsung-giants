# MyProject

MyProject는 팀 협업을 위한 오픈소스 생산성 도구입니다.

## 주요 기능

- 실시간 협업 편집
- 버전 히스토리 자동 저장
- 슬랙·지라 연동

## 설치 방법

```bash
npm install myproject
myproject init
```

## 빠른 시작

```bash
myproject start --port 3000
```
